# Trading-automation-backend
This Repositories contains a demo project used for trading automation, AngelOne api are used here for trading purposes. 

# Install Dependencies
use npm install to install all dependencies.

# Set Environment variables
Please create .env file with all variable shown in .env.example file

# Startup
To start the server just run npm start or node index.js the server will run on port 3002 as default you can update the port in index.js file or added in your environment file
